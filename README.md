# NeighbourHub

NeighbourHub is a community marketplace that allows neighbours to lend and borrow equipment instead of buying expensive tools they only need once.

## Features

- Browse nearby items
- Search items
- Filter items
- Item details
- Booking flow
- Responsive design
- Login page

## Technologies

- React
- TypeScript
- Vite
- Tailwind CSS
- React Router

## Installation

```bash
npm install
npm run dev
```
